## 🚀 ИННОВАЦИОННЫЕ ИНТЕРАКТИВНЫЕ РЕШЕНИЯ

### 1. Процедурный Кинетический 3D-Фон
* **Материнская плата в центре:** Вместо загрузки тяжелых моделей, плата генерируется процедурно в JavaScript. Она переливается насыщенным изумрудным цветом и плавно поворачивается при скролле.
* **Световые эффекты:** Трехточечная неоновая подсветка (Cyber Cyan, Neon Pink, Cyber Green) создает реалистичные переливы и тени на металлических контактах и конденсаторах.
* **Бегущие импульсы:** По золотым дорожкам схемы непрерывно движутся бирюзовые и салатовые светящиеся импульсы данных.

### 2. Динамическое Заполнение Пространства (Responsive Margin Assets)
* **Боковые модули:** Для широких мониторов по краям экрана добавлены парящие в невесомости 3D-модели (микроконтроллер **ESP32**, мини-компьютер **Raspberry Pi** и пальчиковые **AA-батарейки**).
* **Математическая адаптивность:** Координаты парящих элементов рассчитываются на лету по формуле `edgeX = aspect * 3.8`. Они автоматически отодвигаются к рамкам окна на любых мониторах (включая UltraWide 21:9), оставляя центр свободным.
* **Мобильная оптимизация:** При ширине экрана менее `1024px` все боковые элементы плавно сжимаются до масштаба `0` и отключаются, полностью освобождая место для мобильного интерфейса и снижая нагрузку на процессор телефона.

### 3. Полноэкранный Интерактивный Вьювер Документов (Zoom & Pan Viewer)
* **100% Fullscreen:** Модальные окна раскрываются на весь экран с глубоким размытием заднего фона (`backdrop-blur-xl`).
* **Scroll-to-Zoom:** Пользователь может плавно увеличивать изображение колесиком мыши (от **100% до 600%**) для изучения мелкого шрифта на дипломах и сертификатах.
* **Drag-to-Pan:** В увеличенном масштабе зажатием мыши можно легко перетаскивать и двигать изображение в любом направлении.
* **Фиксированный крестик закрытия:** Кнопка закрытия вынесена в фиксированную позицию в верхнем правом углу экрана (`fixed top-6 right-6 z-50`) для максимального удобства.

## ⚡ ИНСТРУКЦИЯ ПО ЛОКАЛЬНОМУ ЗАПУСКУ

Скомпилированные файлы используют современные JS-модули (`ES Modules`), поэтому для корректной работы шрифтов, PDF и 3D-графики файлы должны запускаться через любой локальный веб-сервер (прямой запуск двойным кликом по `index.html` может блокироваться политикой безопасности браузера CORS).

### Вариант 1. Через Node.js (самый простой)
Если у вас установлен Node.js, выполните в терминале папки `dist`:
```bash
npx serve
```
После этого откройте в браузере адрес: `http://localhost:3000`

### Вариант 2. Через Python
Если на компьютере установлен Python, выполните в терминале папки `dist`:
```bash
# Для Python 3
python -m http.server 8000
```
После этого откройте в браузере адрес: `http://localhost:8000`

### Вариант 3. Расширение Live Server (в VS Code)
1. Откройте папку `dist` в редакторе VS Code.
2. Нажмите кнопку **"Go Live"** в правом нижнем углу окна редактора.
